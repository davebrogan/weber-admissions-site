A text-box is a module that provides an h2 title and regular text in either White or University Gray text colors.

The color-background-box is a module that provides a content container with a colored background that can allow another background to show through by lowering the color alpha value. The color of the background is controlled by adding college color theme classes (i.e., "-csbs--bg-trans-7"). Originally designed by Dave Brogan for the President's Office 2018 Annual Report.

A banner provides image content as a background and can hold additional content. It can position content using "--center-content", "--left-content" and "--right-content" modifiers.
